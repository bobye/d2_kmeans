d2_kmeans
=============

D2-Kmeans: accelerated and parallel clustering of discrete distributions

## How to compile

### Linux + Intel Compiler

dependencies

 - MKL
 - openMP support
 - Mosek 7.0+

### Mac OS 10.10 + Clang

dependencies

 - Accelerate Framework
 - no OpenMP support
 - Mosek 7.0+
